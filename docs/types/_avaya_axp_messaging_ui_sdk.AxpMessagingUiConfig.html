The configuration for the AXP Messaging UI SDK.

    The 36 character unique application key.

  • Optional defaultTheme?: string

    The name of the default theme.

  • Optional displayStrings?: DisplayStrings

    The display strings for the AXP Messaging UI SDK. This configuration is used to customize the various display strings used in the AXP Messaging UI SDK and their translations.

  • Optional emojiMartTranslations?: Partial<Record<Locale, EmojiMartTranslation>>

    The map of Locale and the translation for the EmojiMart component in the Messaging UI SDK.

  • host: string

    Hostname of the AXP API endpoint to connect to or an AXP region code. If a region code is provided (e.g., "na" for North America), it will be converted to the corresponding hostname (e.g., "na.api.avayacloud.com"). If a full hostname is provided (e.g., "na.api.avayacloud.com"), it will be used as is.

  • Optional idleShutdownGraceTimeoutDuration?: number

    The duration (in milliseconds) till which user is allowed to remain inactive after the idle timeout. SDK will shutdown on timeout.

  • Optional idleTimeoutDuration?: number

    The duration (in milliseconds) till which user is allowed to remain inactive, after which grace timer will start.

  • integrationId: string

    The 36 character unique integration identifier that is configured on AXP.

  • Optional locale?: Locale

    The locale for the Messaging UI SDK. The locale is used to determine the translations of the Messaging UI. The locale should be in the ISO 639-1 format, example "en-US".

  • Optional logLevel?: LogLevel

    The default log level for the Messaging UI SDK.

  • Optional onIdleTimeOut?: ((instance) => void)

        Returns void

        Returns void

  • Optional onLocationRequest?: LocationRequestHandler

    The handler function that will be called when the request for location is made.

  • Optional onMessageBubbleClicked?: ((instance) => void | Promise<void>)

      • (instance): void | Promise<void>

      • Parameters

        Returns void | Promise<void>

        Returns void
