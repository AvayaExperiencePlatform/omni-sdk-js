Parameters to initialize the SDK and create a session for this SDK instance on the server.

Type declaration

  • appKey: string

    Application Key which is required to access the API for your Account.

  • OptionaldisplayName?: string

    How the user is presented to others.

  • host: string

    Hostname of the AXP API endpoint to connect to or an AXP region code. If a region code is provided (e.g., "na" for North America), it will be converted to the corresponding hostname (e.g., "na.api.avayacloud.com"). If a full hostname is provided (e.g., "na.api.avayacloud.com"), it will be used as is.

  • OptionalidleShutdownGraceTimeoutDuration?: number

    The duration (in milliseconds) till which user is allowed to remain inactive after the idle timeout. SDK will shutdown on timeout.

  • OptionalidleTimeoutDuration?: number

    The duration (in milliseconds) till which user is allowed to remain inactive, after which grace timer will start.

  • integrationId: string

    The 36 character unique UUID for the tenant that is configured on AXP.

  • jwtProvider: JwtProvider

    Interface implemented by applications to access a valid JWT for the SDK using the connection to their own application server.

  • OptionallogLevel?: LogLevel

    Logging level to use for SDK.

  • OptionalsessionParameters?: Record<string, string>

    The key value pairs capturing information about the user's session.

  • token: string

    Initial Jwt to initialize the SDK
